        AQR System Information
        
            
        
            
        
                
        1. Purpose & GxP Scope
        
                
        
                    The Air Quality Review (AQR) Dashboard is a cGMP-compliant computerized software system designed under GAMP 5 Category 5 guidelines. It automates the extraction, compliance review, and reporting of critical cleanroom environmental parameters (Temperature, Relative Humidity, Differential Pressure) captured from Building Automation Systems (BAS).
                
        

                
        2. Operational SOP & Workflows
        
                
          
                    
        1. Prepare CSV Files: If your BAS export is a combined log, navigate to the Data Transformation page and use the Report Splitter to split it into room-specific files.
          2. 
                    
        2. Configure Directory Paths: Select the source mode (Insight/Info format for primary facilities or Desigo format for expanded operations). Select the split CSV directory path and the corresponding Excel setpoint/limits specification file.
          3. 
                    
        3. Execute Quality Review: Run the analyzer to generate high-fidelity plots, heatmaps, and a summary breakdown of violation events (> 20 min thresholds).
          4. 
                    
        4. Fine-tune & Tweak Thresholds: Zoom, pan, or inspect specific room plots dynamically. Interactively test different High/Low limit setpoints in real-time in each plot's header.
          5. 
                
        

                
        3. Temporal Boundary Standard
        
                
        
                    In accordance with regulatory audit requirements and cleanroom logging conventions, file-level date boundaries are parsed strictly from the internal data timestamps to guarantee chronological integrity, mitigating external filename discrepancies.
                
        
            
        
            
        
        
    
        

    
    
        
        
        
            
        
                
        
                    
                
        
                
        About AQR Program
        
                v1.1.0
            
        
            
        
                
        
                    
        
                        System Identity: Air Quality Review (AQR) Validation Dashboard
                    
        
                    
        
                        Secure Cryptographic Audit Trail: The system implements a robust, tamper-evident cryptographic security log (GAMP 5 compliance). All transactions, reports, startup checks, and process failures are timestamped and signed using SHA-256 hashing for absolute data integrity.
                    
        
                    
        
                        WSGI Server Isolation: Served using Waitress WSGI production server to enforce standard security profiles, eliminating standard development server vulnerabilities.
                    
        
                    
        
                        Automated Cache Buster: Powered by the AQR Hot-Reload Engine (v2.2) to dynamically update visual resources on client requests.
