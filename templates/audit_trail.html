About AQR Program

Air Quality Review (AQR) System is designed for high-integrity environmental review (GAMP 5 Category 4/5).

Version: 1.1.0 (Waitress Production WSGI Server)

AQR User Guidance

This module provides an interactive view of the GAMP 5 compliant secure audit log.

  • Tamper-Evident Chain: Each event entry calculates a SHA-256 hash linked to the previous row's hash.
  • Immediate Verification: Run "Verify Trail Integrity" to check for database corruption or tampering.
  • Real-time Filtering: Search for room names, error codes (like `ERR-010`), or actions instantly.
