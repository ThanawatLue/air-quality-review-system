Procedure ในการทดสอบระบบในการ validate ระบบ Advance Monitoring System จะมีการทดสอบทั้งสิ้น 4 stepModule Testing (เป็นการทดสอบเพื่อดูว่า coding ถูกหรือไม่)Result: screen capture ตอนทำ debugIntegration Testing (เป็นการทดสอบดูการ interface ระหว่าง module)Result: screen capture ผลที่ได้ตาม test scriptFunctional Testing (เป็นการทดสอบฟังก์ชันทั้งหมดของระบบ)Result: screen capture ผลที่ได้ตาม test scriptRequirement Testing (เป็นการทดสอบว่าระบบสามารถทำงานได้ตาม requirement หรือไม่)Result: screen capture ผลการทำงานของระบบติดต่อกัน 24 ชม.
